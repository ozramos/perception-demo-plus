# Wayve Perception Demo +
**Learn more**: [https://wayve.ai/blog/2018/10/8/vision-for-driving-with-deep-learning](https://wayve.ai/blog/2018/10/8/vision-for-driving-with-deep-learning)

**Original Demo**: [http://perception.wayve.ai](http://perception.wayve.ai)

**Me**: [https://twitter.com/labofoz](https://twitter.com/labofoz)

This quick demo expands on [Wayve's new SegNet model](https://wayve.ai/blog/2018/10/8/vision-for-driving-with-deep-learning) by inferring off of Google Street Views. You can extend this idea by automating the Street View using only the learned data, reconstructing photo realistic cities in full 3D, and sooooo much more!

The code controlling the street view is in [vroom-vroom.js](https://glitch.com/edit/#!/perception-demo-plus?path=vroom-vroom.js:1:0), along with some simple helpers for extracting the data. Use the context
